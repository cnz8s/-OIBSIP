machine learning

  • Machine learning is a subset of artificial intelligence (AI) that focuses on learning from data and developing an algorithm that may be used to generate a prediction.
  • Machine Learning takes a data-driven method, often training on historical data before making predictions on fresh data.
  • ML can detect patterns and insights in vast datasets that people may struggle to discover.
  • there are types of machine learning they are Supervised Machine Learning,Unsupervised Machine Learning,Semi-Supervised Machine Learning,Reinforcement Learning
